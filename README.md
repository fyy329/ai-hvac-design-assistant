# AI HVAC Design Assistant

This project explores the integration of AI-powered code generation (Codex) with HVAC and building energy system design.

## 🚀 Overview

The goal is to develop a GUI-based assistant that helps engineers:

- Automate simulation setup
- Configure HVAC system parameters
- Optimize energy system performance
- Reduce manual modeling effort

## 🧠 Motivation

Modern HVAC and energy system design often involves repetitive and complex workflows.  
This project aims to leverage AI to streamline engineering processes and improve productivity.

## 🛠️ Planned Features

- GUI for HVAC system configuration
- Codex integration for automated code generation
- Simulation workflow automation (e.g. Polysun / Modelica)
- Parameter optimization tools
- Data-driven design support

## 📌 Status

Early-stage development. Initial structure and concept validation.
